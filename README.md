## Comandos para construir e subir uma máquina virtual no Docker

Build:

```sh
docker-compose --file compose.yaml build
```

Run:

```sh
docker-compose --file compose.yaml up -d
```

Stop:

```sh
docker-compose --file compose.yaml down
```
